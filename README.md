# wayneshop

您好，

歡迎來到 Wayne Lu 的練習購物網站，

本專案僅作為練習，無商業用途，

# 使用技術

利用 Vue.js 搭配 Vue CLI 建立專案

使用 scss 做顏色變數管理，

使用 Vue3 和 Pinia 構建購物車系統，

並使用 pinia-plugin-persistedstate 實現購物車內數據存到 LocalStorage 達到數據保存，

使用 elementPlus 設置提示訊息及處理表單驗證，

使用 vueuse 實現更多更能互動，

利用 axios 處理 AJAX 請求，獲取後端資料實現登入功能，
